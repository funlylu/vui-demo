# san-player

## 安装依赖
```
npm run init
```

### 运行示例
```
npm run dev
```

### 打包示例demo
```
npm run build
```

### 代码校验
```
npm run lint
```

### 打包生成 ui 组件库文件
```
npm run lib
```

### 生成一个组件模板框架
```
npm run new:comp
```
### 删除一个组件模板框架
```
npm run del:comp
```

### 规范提交 git commit 根据提示输入相关日志内柔
```
npm run commit
```

### 整个生成 changelog.md 
```
npm run initlog
```
### 追加 changelog.md 文档
```
npm run addlog
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).


## 目录结构说明

examples —— 是组件示例展示demo界面
packages —— 组件库核心代码
lib —— 组件库打包后的代码
scripts—— 工具文件——生成组件、删除组件 以及模板文件
